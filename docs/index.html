Cancer Subtype Discovery using Unsupervised Learning

This site presents the results of a bioinformatics project applying machine learning to gene expression data.

Project Summary

  • Dimensionality reduction using PCA, t-SNE, UMAP
  • Clustering using KMeans, DBSCAN, Hierarchical
  • Subtype discovery and evaluation via Adjusted Rand Index
